## 🎓 Technical Implementation

### Algorithm Components

1. **Multi-Indicator Detection**
   ```
   Find ALL matching stress indicators (not just max)
   ├─ Critical level (0.95)
   ├─ High stress (0.6-0.65)
   ├─ Moderate (0.4-0.5)
   ├─ Mild (0.1-0.3)
   └─ Positive (-0.2 to -0.25)
   ```

2. **Weighted Averaging**
   ```
   Balance multiple indicators by severity
   weight = sum(value * abs(value)) / sum(abs(value))
   ```

3. **Frequency Boosting**
   ```
   Multiple mentions increase stress
   boost = min(0.15, (count - 1) * 0.05)
   ```

4. **Non-Linear Scaling**
   ```
   Convert 0-1 probability to 0-100 health score
   Uses variable-range mapping for realism
   ```
